# 💅 The Nail Masters

A modern, luxury, and fully responsive website for **The Nail Masters**, a premium nail and beauty studio located in Mohali, India.

Built with a focus on elegant UI/UX, performance, scalability, and SEO, this project provides customers with a seamless experience to explore services, view the gallery, and book appointments online while giving administrators a powerful dashboard to manage the business.

## ✨ Features

- Luxury responsive UI
- Online appointment booking
- Service & pricing management
- Gallery with image optimization
- Testimonials & FAQs
- Contact & WhatsApp integration
- Google Maps integration
- Admin Dashboard
- Authentication & Authorization
- MongoDB Database
- Cloudinary Image Uploads
- SEO Optimized
- Fast page loading
- Mobile-first design

## 🛠 Tech Stack

### Frontend
- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI
- Framer Motion
- GSAP
- Lenis Smooth Scroll

### Backend
- Node.js
- Express.js
- MongoDB
- Prisma ORM
- JWT Authentication
- Nodemailer
- Cloudinary
